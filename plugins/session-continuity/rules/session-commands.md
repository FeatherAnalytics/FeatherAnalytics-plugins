# Session Continuity Commands

When the user asks to do any of the following, invoke the corresponding command:

- "create a handoff", "hand off work", "transfer context", "preserve session" → `/create-handoff`
- "resume handoff", "pick up where I left off", "continue from handoff", "load handoff" → `/resume-handoff`
- "make a checkpoint", "save this checkpoint", "checkpoint this session", "save state", "save progress" → `/checkpoint`
- "summarize session", "summarize work stream", "session summary", "session diary", "capture session" → `/summarize-session`
